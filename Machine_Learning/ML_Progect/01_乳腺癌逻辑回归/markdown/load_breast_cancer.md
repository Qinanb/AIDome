# sklearn 乳腺癌数据集（load_breast_cancer）介绍

## 1. 数据集概述
`load_breast_cancer` 是 scikit-learn 内置的经典**二分类数据集**，用于乳腺癌良性/恶性诊断预测。
该数据集基于**细胞细胞核的数字化影像特征**构建，是机器学习入门分类任务的标准数据集。

## 2. 数据集来源
- 原始数据来自 **UCI 机器学习库**
- 由威斯康星大学医学院整理
- sklearn 对其进行了预处理，方便直接使用

## 3. 数据集基本信息
- **样本数量**：569 个
- **特征数量**：30 个数值型特征
- **任务类型**：二分类任务
- **目标变量**：肿瘤类型（良性 / 恶性）

## 4. 特征说明
所有特征均为**细胞核的量化特征**，包括：
- 半径（半径平均值）
- 纹理（灰度标准差）
- 周长
- 面积
- 光滑度
- 紧凑度
- 凹度
- 凹点
- 对称性
- 分形维度

每个特征包含 **平均值、标准误差、最大值** 三类统计量，因此总特征数为 10 × 3 = 30 个。

## 5. 目标变量（标签）
- `0`：**恶性（Malignant）**
- `1`：**良性（Benign）**

## 6. 数据集特点
- 无缺失值
- 全部为数值特征
- 类别分布略微不平衡（良性样本更多）
- 适合测试分类算法（逻辑回归、SVM、决策树、神经网络等）

## 7. 数据集导入方法（Python）
```python
from sklearn.datasets import load_breast_cancer

# 加载数据集
cancer = load_breast_cancer()

# 特征与标签
X = cancer.data  # 特征矩阵 (569, 30)
y = cancer.target  # 标签向量 (569,)

# 查看信息
print("特征名称：", cancer.feature_names)
print("标签名称：", cancer.target_names)
print("数据形状：", X.shape)
