# sklearn.make_blobs 人工聚类数据集详解
## 一、数据集概述
`sklearn.datasets.make_blobs` 是Scikit-learn提供的**人工数据生成函数**，用于快速创建多簇结构化数据，是机器学习中测试聚类、分类算法的标准测试数据。

## 二、核心功能
生成服从高斯分布的多簇样本点，特征维度、簇数量、离散度均可自定义。

## 三、关键参数
| 参数名 | 类型 | 默认值 | 说明 |
|--------|------|--------|------|
| n_samples | int | 100 | 总样本数量 |
| n_features | int | 2 | 样本特征维度 |
| centers | int/array | 3 | 簇数量或簇中心坐标 |
| cluster_std | float/list | 1.0 | 簇的离散程度 |
| random_state | int | None | 随机种子，保证数据可复现 |

## 四、数据格式
- **特征矩阵X**：`(n_samples, n_features)` 二维数组
- **标签向量y**：`(n_samples,)` 一维数组，取值为0,1,2...

## 五、典型使用代码
```python
from sklearn.datasets import make_blobs
import matplotlib.pyplot as plt

# 生成数据
X, y = make_blobs(n_samples=500, n_features=2, centers=3, random_state=42)

# 可视化
plt.scatter(X[:,0], X[:,1], c=y, cmap='viridis', edgecolor='black')
plt.title('Clustering Data Generated by make_blobs')
plt.xlabel('Feature 1')
plt.ylabel('Feature 2')
plt.show()
